# Free Fire Sensitivity Website

## Setup Instructions

### Adding Sensitivity Images

1. Place your sensitivity setting images in the `images` folder
2. Name them exactly as follows:
   - `samsung-sensi.jpg`
   - `iphone-sensi.jpg`
   - `iqoo-sensi.jpg`
   - `redmi-sensi.jpg`
   - `realme-sensi.jpg`
   - `oneplus-sensi.jpg`

### Image Requirements
- Format: JPG, PNG, or WEBP
- Recommended size: 800x1200px or similar aspect ratio
- Clear screenshots of Free Fire sensitivity settings

### How to Use
1. Open `index.html` in your browser
2. Click on any device to view its sensitivity settings
3. Download the image using the download button
4. Click "Back to Devices" to select another device

### Customization
- Edit `script.js` to change image paths or add more devices
- Modify `style.css` to change colors and styling
- Update `index.html` to add/remove devices
